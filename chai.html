under the hood

Three steps. Zero magic.

🔍
01 — Scan

DOM traversal

Uses querySelectorAll('[class*="chai-"]') to find every element with a chai-* class. MutationObserver catches dynamic additions.

⚙️
02 — Parse

Class → Style object

Six modular parsers (colors, spacing, typography, borders, layout, shadows) each handle their category and return a plain JS style object.

✨
03 — Apply

Inline styles + cleanup

Styles are merged and applied via element.style. The chai-* classes are then removed, keeping the DOM clean.

DOM Scanner → Class Parser → Style Applier → Clean DOM
